# wannabe_rag_assignment 🤪
 
A lightweight, browser-based attempt at mimicking Retrieval-Augmented Generation (RAG) behavior using plain JavaScript — no backend required. Upload a PDF, get a summary of its contents, and ask questions about what's inside it.

## What it does
 
- **Upload a PDF** directly in the browser
- **Parse and extract text** from the PDF using [pdf.js](https://mozilla.github.io/pdf.js/)
- **Summarize** the extracted content
- **Ask questions** about the PDF and get answers derived from its contents
This project is a simplified, client-side simulation of the retrieve-then-generate pattern used in real RAG pipelines — hence the "wannabe" in the name.
